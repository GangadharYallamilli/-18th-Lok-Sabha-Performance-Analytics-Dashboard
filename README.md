9# 18th Lok Sabha Performance Analytics Dashboard

## Project Overview

This project presents a **data-driven analysis of the 18th Lok Sabha (2024–2029)** using real parliamentary records.
The dashboard provides insights into the performance of Members of Parliament (MPs) by analyzing participation, debates, questions, and other parliamentary activities.

The project uses **Power BI** to create an interactive dashboard for exploring parliamentary data from **542+ MPs**.

---

## Technologies Used

* Power BI
* Microsoft Excel
* Data Cleaning & Transformation
* Data Visualization

---

## Dataset

The dataset contains records of **542+ Members of Parliament** from the **18th Lok Sabha**, including:

* MP Name
* Party Affiliation
* State / Constituency
* Questions Asked
* Debates Participated
* Attendance / Participation
* Parliamentary Activity Metrics

---

## Dashboard Features

The Power BI dashboard includes:

* MP performance comparison
* State-wise parliamentary activity analysis
* Party-wise performance insights
* Participation in debates and questions
* Interactive filters and slicers for better exploration

---

## Key Insights

This dashboard helps analyze:

* Which MPs are most active in Parliament
* State-wise and party-wise participation trends
* Parliamentary engagement patterns across India

---

## Project Goal

The goal of this project is to use **data analytics and visualization** to make parliamentary performance more transparent and understandable.
